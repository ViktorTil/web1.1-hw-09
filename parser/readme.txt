Пакет для загузки парсинга сайта: http://quotes.toscrape.com/, выборки из
него коллекций authors и quotes, формирования из них json-файлов, выгрузки их 
в БД(локальную) в виде коллекций authors и quotes и обработки скриптов запросов
в бесконечном цикле:

main.py - главный модуль(ГМ) запуска - распарсит сайт, создаёт json-файлы,
формирует и заполняет базу данных из json в локальное облако и обрабатывает
скрипты запросов в бесконечном цикле

scraper.py - модуль парсинга сайта: http://quotes.toscrape.com/, выборки из
него коллекций authors и quotes, формирования из них json-файлов:
authors.json и quotes.json.(запускается из ГМ)

models.py - модуль с моделями json-файлов для БД authors и quotes.
connect.py - модуль соединения к БД
create.py - модуль для создания и заполнения БД из json-файлов(вызывается в ГМ).
data.py - файл с данными: название сайта, имена файлов для создания, командами для скрипта.
